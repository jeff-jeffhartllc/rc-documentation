# About this guide

**Audience:** Daily users and analysts  
**Apps:** REGIS APP (corporate) and REGIS FRANCHISEE APP (franchisee)

This guide covers **day-to-day use** of the Regis Domo applications on https://regiscorp.domo.com — navigation, filters, pages, exports, and sharing.

## Which app should I use?

| App | Who | URL |
| --- | --- | --- |
| **REGIS APP** | Corporate users | https://regiscorp.domo.com/app-studio/183500481/pages/925282956 |
| **REGIS FRANCHISEE APP** | Franchisee users (store-scoped) | https://regiscorp.domo.com/app-studio/2028360971/pages/1195391822 |

Franchisee users see only their own stores. Corporate users see the full organization.

## How to navigate this guide

- Use **Contents** at the front of this document (Ctrl+click / Cmd+click a topic).
- Or use **View → Navigation pane** in Microsoft Word.
- **REGIS APP** topics are in the corporate section; **REGIS FRANCHISEE APP** topics follow.

For system administration, data pipelines, PDP configuration, and runbooks, open **Regis-Admin-Guide.docx**.

## Getting help

If you cannot access an app, see missing data, or need a change to cards or reports, contact your Regis Domo administrator. Escalation contacts are listed in the Admin Guide.
