## Purpose

Monthly store-level report card with cumulative scores and same-store sales metrics for your salons.

## Opening the page

REGIS FRANCHISEE APP → **Store Performance Report Card**  
https://regiscorp.domo.com/app-studio/2028360971/pages/1731862460

![Store Performance Report Card — franchisee app](../../assets/app-2028360971-page-store-performance-report-card.png)

## Filters

| Filter | Notes |
| --- | --- |
| Brand, Salon | Scope to brands or salons on your account |
| Country, State / Province, Territory, DMA | Geographic |
| Is Active? | Active Only |
| Distribution Pattern | Normal Only (common default) |
| Filter by Month | Reporting month for the report card |

## Key cards

| Card | What it shows |
| --- | --- |
| Report Card | Monthly store scores |
| Avg Cumulative Score — In Scope | Average score across in-scope salons |
| SSS — MTD / QTD | Same-store sales metrics |

Letter-grade thresholds are on the **Reference** page.
