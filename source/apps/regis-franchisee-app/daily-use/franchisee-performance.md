## Purpose

**Franchisee Performance** is the default landing page for franchisee users. It shows sales, traffic, location, productivity, loyalty, and digital engagement KPIs scoped to your assigned salons via PDP.

## Opening the page

1. Open REGIS FRANCHISEE APP (default landing page).  
   https://regiscorp.domo.com/app-studio/2028360971/pages/1195391822
2. **Expected result:** KPI dashboard loads with your franchisee's data.

![Franchisee Performance — franchisee app](../../assets/app-2028360971-page-franchisee-performance.png)

## Key differences from corporate Franchisee Performance page

| Aspect | REGIS APP (corporate) | REGIS FRANCHISEE APP |
| --- | --- | --- |
| Data scope | All franchisees (filter to select) | Your franchisee only (PDP) |
| Entity Type filter | Present | Absent |
| Location map | Active Regis Locations | Active Franchise Locations |
| Rollup default | Brand or DMA | DMA |

## Card sections

Same sections as the corporate performance pages:

- **Sales** — Total Sales, Service Sales, Retail Sales, AUV, trends
- **Traffic** — Total Traffic, heatmaps, service/retail/combo splits
- **Locations** — Active Salon Count, top/bottom performers, store counts
- **Productivity** — SPH, productive hours, cuts/guests per hour
- **Products & Services** — Retail attach, combo %, color %
- **Loyalty & Retention** — 90/180-day retention, loyalty sales %
- **Digital Engagement** — Email capture, online booking

## Common tasks

### Task: Review month-to-date performance

1. Set **From Date** to first of month, **Through Date** to today.
2. Confirm **Is Active?** = Active Only.
3. Review **Total Sales**, **Total Sales Trend**, and **AUV**.
4. **Expected result:** MTD metrics for your salons.

## If this fails

| Symptom | Action |
| --- | --- |
| All cards blank | See [PDP troubleshooting](../maintenance/pdp-troubleshooting.md) |
| Wrong salon count | Escalate to PDP / access owner |
| Stale data | Contact support — likely a data refresh issue |
