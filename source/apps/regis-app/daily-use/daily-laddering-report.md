## Purpose

The **Daily Laddering Report** tracks day-over-day operational metrics including service sales per hour, service guests per hour, comp salon count, and salon closures. This page is **corporate-only**.

## Opening the page

1. REGIS APP → **Daily Laddering Report**

## Common tasks

### Task: Review today's laddering metrics

1. Open the page with default filters (Active Only, Normal Only).
2. Review the laddering table for SPH, guest counts, and closure counts.
3. Compare day-over-day progression.
4. **Expected result:** Current day metrics with prior-day comparison columns.
